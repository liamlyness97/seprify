# Seprify

A demo build for Alt Design

## Tech Stack

**Client:** TailwindCSS, Svelte, Sveltekit, Motion One

## Run Locally

Clone the project

```bash
  git clone https://github.com/liamlyness97/seprify
```

Go to the project directory

```bash
  cd seprify
```

Install dependencies

```bash
  pnpm install
```

Start the server

```bash
  pnpm dev
```
